📘 Logarytmy

Definicja

loga(b) = c ↔ ac = b

Przykład: log2(8) = 3, bo 23 = 8

Wzory

loga(MN) = loga(M) + loga(N)

Logarytm z iloczynu to suma logarytmów – używamy, gdy mnożysz liczby w logarytmie.

loga(M/N) = loga(M) - loga(N)

Logarytm z ilorazu to różnica logarytmów – gdy dzielisz liczby w logarytmie.

loga(Mk) = k · loga(M)

Potęga „spada” przed logarytm – bardzo przydatne przy równaniach.

loga(a) = 1

Zawsze 1, bo a1 = a

loga(1) = 0

Zawsze 0, bo a0 = 1

Zastosowanie

Logarytmy przydają się do przekształcania wyrażeń, rozwiązywania równań wykładniczych oraz w zadaniach z funkcją wykładniczą.

Przykładowe zadanie z rozwiązaniem

Zadanie: Oblicz log3(1/27)

Krok 1: Zapisz 1/27 jako potęgę 3 → 1/27 = 3-3

Krok 2: log3(3-3) = -3

Odpowiedź: -3 (bo potęga weszła „przed” logarytm)

Zadania z matury

  • Matura 2021: Oblicz log3(1/9) → Odp: -2
  • Matura 2018: log5(x) = 2 → x = 25
  • Matura 2016: log4(64) → Odp: 3

📌 Zapamiętaj

  • loga(1) = 0
  • loga(a) = 1
  • Nie logarytmujemy liczb ≤ 0
