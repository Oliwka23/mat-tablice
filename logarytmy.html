Logarytmy

📘 Definicja

loga(b) = c ↔ ac = b

Gdzie: a > 0, a ≠ 1, b > 0

🧠 Własności logarytmów

  • loga(MN) = loga(M) + loga(N)
  • loga(M/N) = loga(M) - loga(N)
  • loga(Mk) = k · loga(M)
  • loga(a) = 1,   loga(1) = 0
  • Zmiana podstawy: loga(b) = logc(b) / logc(a)

🛠️ Zastosowanie

Logarytmy pojawiają się na maturze przy rozwiązywaniu równań, upraszczaniu wyrażeń oraz w zadaniach z wykładniczością.

📌 Przykładowe zadanie

Oblicz: log2(8)

Rozwiązanie: 23 = 8, więc log2(8) = 3

📝 Zadania z matury

  • Matura 2021: Oblicz log3(1/9) → Odp: -2
  • Matura 2018: Dla jakiego x: log5(x) = 2 → x = 25
  • Matura 2016: Oblicz log4(64) → Odp: 3
